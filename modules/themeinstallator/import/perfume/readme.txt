Prestashop Installation Guide

Author  : loftytemplates
Email   : loftytemplates@gmail.com


Procedure for upload and install the Prestashop:

	Step 1: Upload the Perfumetheme folder to your prestashop 

	Step 2: Login to your prestashop admin panel and activate Perfume theme
	
	Step 3: Goto Images under Preferences and set the image sizes 
	
				      
                        Home_default		Width: 173px, Height: 186px;

        Step 4: Goto Modules-->Modules-->Search Image slider-->click Configuration  and change 

                width as 958px and height as 385px
     
        Step 5: Goto Module--> Position-->Transplant a Module-->select category Module in Right side Column Hook
   
        Step 6: Goto Module--> Position-->Transplant a Module-->select Cart Module in Right side Column Hook
 
        Step 7: Goto Module--> Position-->Right Column-->Delete New Products,Special,Contactus
        
        Step 8: Goto Module--> Position-->Header Column-->Delete Cartblock
